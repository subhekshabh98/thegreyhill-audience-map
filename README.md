## Project Overview
This project visualises Scottish theatre audience data for The Greyhill using a map-based web application.

## Features
- Excel upload
- Data parsing
- Geocoding addresses
- Plotting audience on a map
- Filtering by show

## Folder Structure
backend/ – Node.js server  
frontend/ – Upload + map UI  
data/ – Sample ticketing datasets  
docs/ – Project plan & documentation

## How to Run
1. cd backend
2. npm install
3. node server.js
4. Open frontend/index.html in browser
